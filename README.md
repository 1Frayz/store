Это проект Laravel, развернутый с использованием Docker. Данное приложение включает сервисы PHP-FPM, Nginx и MySQL.

## Начало работы

### Требования

- Docker
- Docker Compose

### Установка

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/1Frayz/store.git
   cd store
   ```

2. Скопируйте файл `.env.example` и создайте файл `.env`:

   ```bash
   cp .env.example .env
   ```

3. Настройте параметры в файле `.env` при необходимости.

4. Запустите контейнеры:

   ```bash
   make start
   ```

5. Сгенерируйте ключ приложения:

   ```bash
   make key-generate
   ```

6. Выполните миграции:

   ```bash
   make migrate
   ```

7. Откройте в браузере [http://localhost](http://localhost).

## Команды Makefile

- `make start` - Запускает контейнеры.
- `make stop` - Останавливает контейнеры.
- `make restart` - Перезапускает контейнеры.
- `make down` - Останавливает и удаляет контейнеры.
- `make key-generate` - Генерирует новый ключ приложения.
- `make migrate` - Выполняет миграции базы данных.
- `make fresh` - Выполняет свежую установку базы данных с сидированием.

## Структура проекта

- `app/` - Содержит ядро приложения.
- `bootstrap/` - Содержит файлы начальной загрузки и кеширования.
- `config/` - Содержит все конфигурационные файлы.
- `database/` - Миграции базы данных и сидеры.
- `public/` - Публичные файлы, такие как index.php и ресурсы.
- `resources/` - Шаблоны и локализации.
- `routes/` - Файлы маршрутов.
- `storage/` - Файлы, требующие постоянного хранения, такие как логи и сессии.
- `tests/` - Тесты приложения.
- `vendor/` - Пакеты, установленные через Composer.
